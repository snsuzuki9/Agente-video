# 🎬 Agente de Vídeos Motivacionais

Automação completa para criação de vídeos estilo TikTok usando IA.

## 🚀 Funcionalidades

- Geração de roteiro com IA (Llama3)
- Geração de áudio (Piper TTS)
- Busca de imagens (Pexels API)
- Montagem de vídeo (FFmpeg)
- Legendas profissionais (.ASS)

## 🧠 Fluxo

IA → roteiro → temas → imagens → áudio → vídeo → legenda → render final

## ⚙️ Requisitos

- Node.js
- FFmpeg
- Piper TTS
- API Key Pexels

## ▶️ Como rodar

```bash
npm install
node index.js
